**Использование линз для работы с коллекциями**

Для работы с коллекциями используются `Traversals` и `Folds`. Они позволяют применять операции ко всем элементам коллекции или к конкретному индексу. Так можно работать не только с коллекциями, в которых лежат сложные объекты, но и с примитивными, но для этого лучше использовать стандартные средства.

Основными комбинаторами для коллекций являются:
- `each` - работает с типами реализующим класс `Traversable` и  перебирает все элементы,
- `traverse` - аналог `each`, но требует контекст `Applicative`,
- `ix n` - доступ по индексу - возвращает `Traversal'`, который сработает только если индекс существует,
- `filtered p` - фильтрует элементы коллекции по предикату `p`.

Если у нас есть список `Rectangle` (из предыдущего поста про линзы):
```haskell
let rects = [ Rectangle (Point 1 1) (Point 10 10),
              Rectangle (Point 2 2) (Point 20 20),
              Rectangle (Point 3 3) (Point 30 30),
              Rectangle (Point 4 4) (Point 40 40)
            ]
```

тогда можно, изменить координату `x` в точке `p1` для каждого прямоугольника c помощью `%~` (`over`):
```haskell
rects & traverse . p1 . x %~ (+5) 
      & traverse . p2 . x %~ (+5) 
```

получить список всех значений `x` с помощью `^..` (`toListOf`):
```haskell
let xs = rects ^.. each . p1 . x
      <> rects ^.. each . p2 . x 
```

установить одно значение для всех элементов через `.~` (`set`):
```haskell
let rects4 = rects & each . p1 . x .~ 666
```

Если нужно совершать махинации не со всеми элементами коллекции, то можно использовать фильтрацию (`filtered`) или доступ по индексу (`ix`):
```haskell
 rects3 = rects
        & each
        . p1
        -- на этом уровне в нас Point 
        -- и в filtered придёт именно Point 
        . filtered (\p -> mod (p ^. x) 2 == 0)
        . x %~ (+5)
```

Для агрегации коллекций существуют функции:
- `anyOf` - есть ли хоть один элемент, который удовлетворяет условию,
- `noneOf` - ни один из элементов не удовлетворяет условию, 
- `allOf` - все ли элементы удовлетворяют условию,
- `productOf` - перемножение элементов,
- `sumOf` - суммирование значений,

```haskell
let anyX = anyOf (each . p1 . x ) (>3) rects
let sumX = sumOf (each . p1 . x) rects
```
