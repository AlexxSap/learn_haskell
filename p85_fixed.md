**Вычисления с фиксированной точность**

Иногда для выполнения расчётов в числах с плавающей точкой (`Float` и `Double`) нужно поддерживать заданный уровень точности, чтобы округление малых долей не влияло на расчёт. Для этого нужно использовать модуль `Data.Fixed`, который позволяет задать точность для числа в количестве знаков после запятой от 0 до 12.
```haskell
ghci> import Data.Fixed
ghci> 3.1415926 :: Milli
3.141
```

При этом часть числа, которая нас не интересует отбрасывается, а не округляется. 
Если мы решили создать такое ограничение для работы с числами в нашей программе, то нужно реализовать класс:
```haskell
class HasResolution a where
    resolution :: p a -> Integer
```

Пример:
```haskell
module Main (main) where

import Data.Fixed

data E4
instance HasResolution E4 where
  resolution _ = 10000

type Fixed4 = Fixed E4 

main = do
  print ((33.12345 + 1.00001) :: Fixed4) -- 34.1234
  print ((33.12345 :: Fixed4) + 1.0000999) -- 34.1234
```
Мы создаём новый тип - `Fixed` параметризованный `E4`. Далее его можно использовать как обычный тип для работы с числами.
